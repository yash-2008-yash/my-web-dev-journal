# Chapter 84 — Spotify Clone

This chapter was a major milestone in the course, building a **fully responsive Spotify Web Player clone** from scratch using only HTML, CSS, and Vanilla JavaScript.

This project lives in its own dedicated repository.

## [View the Spotify Clone Repo](https://github.com/yash-2008-yash/Spotify_Clone)

## What was built

A functional Spotify-like music player featuring:

- Responsive UI closely replicating Spotify's web player
- Multiple album playlists with real audio playback
- Play / Pause, Next / Previous track controls
- Seekable progress bar
- Dynamic song info and album art updates
- Clean folder structure separating songs, SVGs, and scripts

## What was learned

- Structuring a large front-end project cleanly
- DOM manipulation and event handling in Vanilla JS
- CSS Flexbox/Grid for complex, responsive layouts
- Working with the HTML5 `<audio>` API
- Managing state (current song, playback status) without any framework
- Organizing assets and keeping code readable at scale
