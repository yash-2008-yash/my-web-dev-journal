# Chapter 101 — Twitter / X UI Clone

This chapter marked the first real **Tailwind CSS project** in the course, building a pixel-close clone of the Twitter (X) web interface using Tailwind as the *only* styling tool. **No plain CSS. No Bootstrap. 100% Tailwind CSS**.

This project lives in its own dedicated repository.

## [View the Twitter / X UI Clone Repo](https://github.com/yash-2008-yash/Twitter_UI_Clone)

## What was built

A static but faithful replica of the X (Twitter) web UI featuring:

- Three-column layout — left nav, main feed, right sidebar
- Native dark theme of X
- Fully responsive structure using Tailwind's breakpoint prefixes
- All styling done exclusively with Tailwind CSS utility classes

## What was learned

This project was a hands-on deep dive into Tailwind CSS:

- Setting up Tailwind CSS via Node/npm (input.css → output.css workflow)
- Utility-first CSS — composing complex components from small, single-purpose classes
- Flexbox & Grid with Tailwind — `flex`, `grid`, `gap-*`, `items-*`, `justify-*`
- Responsive design — `sm:`, `md:`, `lg:` breakpoint prefixes
- Building a full dark UI using Tailwind's color palette
- Getting genuinely comfortable reading and using the Tailwind CSS documentation
